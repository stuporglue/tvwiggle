Wiggle CSS

A css file and class to make stuff wiggle similar to how the new AppleTV show icons wiggle.

The wiggle animation is not very refined here, more of a proof of concept, but it could be smoothed out if needed.

It's css3 and may not work in all browsers. I've been testing/developing in Firefox, Chrome and Safari on OSX.

See also list.html for an example using an unordered list.
